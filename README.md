# Bus Seat Booking Application

A simple full-stack bus seat booking system developed for internship project work.  
This application allows passengers to view bus routes, check available seats, select seats, and confirm bookings.  
It also provides admin features to manage schedules and bookings.

## Purpose
The purpose of this system is to make bus seat booking easy for passengers and help bus operators manage bookings without confusion.

## Main Users
- Passengers
- Admin / Bus company staff

## Main Features

### Passenger Features
- View bus routes
- View available travel dates and times
- View seat layout
- Select available seat
- Enter passenger details
- Confirm booking
- Receive booking ID

### Admin Features
- Add bus schedules
- View all bookings
- Reserve seats
- Cancel bookings

## Functional Requirements
- Display bus routes
- Display available dates and times
- Show seat layout for each bus
- Allow users to select seats
- Prevent double booking of the same seat
- Store passenger details
- Generate unique booking ID
- Allow admin to view all bookings

## Non-Functional Requirements
- Simple and easy-to-use interface
- Fast booking process
- Reliable seat availability updates
- Secure storage of user details

## Technology Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
- Version Control: Git and GitHub
